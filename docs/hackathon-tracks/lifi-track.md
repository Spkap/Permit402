[li.fi](https://li.fi)

### About

LI.FI allows cross-chain swaps and bridging across 60+ chains through a single API — aggregates all major Solana DEXs and 20+ bridges.

### Prizes

🌉 Best Cross-Chain Solana UX powered by LI.FI$1,000

🥇1st place$500

🥈2nd place$300

Build a Solana-relevant application, workflow, or AI agent that uses LI.FI to improve cross-chain onboarding, swaps, bridging, payments, or transaction execution.

LI.FI Widget

Best for teams that want a fast, polished bridge/swap interface inside their application. Add cross-chain and on-chain swap functionality with a prebuilt UI that can be customized to match your product.

LI.FI SDK or REST API

Best for teams that want more control over routing, transaction execution, custom UI, or backend logic. Suitable for wallets, DeFi apps, payments, onboarding flows, and more advanced product experiences.

LI.FI MCP Server / Agent Tooling

Best for AI-agent projects. Build agents that query routes, prepare swaps, guide users through cross-chain actions, or help users understand available Solana liquidity paths using LI.FI-powered tools.

- ◆Cross-chain onboarding into a Solana app: allow users to start from assets on another chain and arrive in the correct Solana asset for the app
- ◆Solana payment or checkout flow: let users pay from any supported chain while the merchant or app receives value on Solana
- ◆AI DeFi assistant: an agent that helps users discover, compare, and prepare Solana-related swap or bridge flows through LI.FI
- ◆Creator or gaming app funding: let users fund a Solana-based experience from non-Solana assets with minimal friction
- ◆Embedded swap/bridge UX for Solana DeFi: use LI.FI routing to simplify how users enter or exit a Solana protocol

- ◆Public GitHub repo
- ◆README explaining the product idea, user flow, and exactly how LI.FI is integrated
- ◆Working demo or clear recorded walkthrough that makes the experience simpler for users
